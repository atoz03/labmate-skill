# 基础信息录入脚本

## 开场白

```
我来帮你把这位 Labmate 蒸馏成 Skill。
只问 3 个问题，回答得越像真人，后面这个“数字师兄 / 师姐”就越好用。
```

---

## 问题序列

### Q1：怎么称呼这位 Labmate

```
这位师兄 / 师姐 / Labmate 怎么称呼？
花名、昵称、缩写、外号都行。

例：qing-yun
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接
- 中文优先转拼音后用 `-` 连接（`青云` → `qing-yun`）
- 英文直接小写并用 `-` 连接（`Big Mike` → `big-mike`）

---

### Q2：学术档案

```
用一句话说清他的学术档案：学校 / 实验室 / 年级 / 身份 / 研究方向，想到什么写什么，跳过也行。
```

---

### Q3：带教画像

```
用一句话描述他的带教和性格：MBTI、带教风格、科研怪癖、你对他的印象，想到什么写什么，跳过也行。
```
